Process of Explore the Wilderness website

The Expolore the Wilderness website was completed as part of my Advanced Projects Studio course. The project invovled multiple steps as I have indicated below.

Steps in completing website

  • Proposal and Management Plan outlining goals, schedule and technologies to be used
  • Market Research on similar websites
  • Creating content and selecting images for project.
  • Mood Board displaying color themes and images.
  • Wireframes
  • Multiple sample style tiles to determine the best design style for the website.
  • HTML and CSS
  • Responsive Design and animations
  • Met with professor frequently to discuss progress on the project
  • Submitted final project on Github
